# Sprint Dynasty - Game Design (MVP)

## Core loop
- 1 turn = choose training / rest / massage
- Meets have Qualifier + Final (8 runners)
- Player fatigue affects race performance (no race fatigue added)
- Entry announcement happens 3 turns before each meet

## Naming rules
- Player name: user input
- JP mob: common surname + given name
- Rare: fancy surname + fancy given name
- Asia/World: katakana full name with "・"
- Superhuman: reminiscent but not real athletes

## Schools
- School name: real Japanese mountain + 高校
- Fixed powerhouse schools appear every time in high-level meets (national+)
- Same-grade rival always exists in major meets
